# reslister
List assets from VMF - pakrat / VIDE resource parsing replacement in Lua 

### Usage
`Usage: reslister [--format=bspzip] "c:\mymap\map.vmf" "c:\mymap\resources" "c:\mymap\reslist.txt"`
